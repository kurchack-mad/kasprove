What is a digital signature?

A digital signature in this context is a SHA-256 hash of a file or text.

What do you mean by "prove"?

KasProve proves that the hash provided by the user matches the payload of the specified transaction provided by the specified archival node.

How can I adapt this to my own project?

Fork the repository on GitHub and modify the code to suit your needs. KasProve is intended to be forked and adapted by software service providers.
